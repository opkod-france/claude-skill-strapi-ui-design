## What's Included

A comprehensive Claude Code skill for Strapi v5 admin UI design:

- **Design System v2** — 47+ compound components
- **Page Layouts** — `Main`, `Layouts.Header`, `Layouts.Content`
- **Data Tables** — Sorting, selection, pagination
- **Forms** — `Field.Root` compound pattern with validation
- **Modals & Dialogs** — `Modal.Root`, `Dialog.Root`
- **Settings & Dashboards** — Tabbed settings, statistics cards
- **Content Manager Integration** — Injection zones, `useFetchClient`
- **Context7 Integration** — Live documentation verification against latest Design System docs
- **Accessibility** — WCAG-compliant, keyboard-navigable
